## Cool Flashcard App

## Lab 4

### App Description
A set of flashcards can be created by user with three answer choices, out of which only one is right.
These flashcards can be edited and saved in the database.
They can also be removed permanently from the application.
Added animations, more styling, Konfetti when right answer is selected, and an icon for the app.

## Lab 2

### App Description
User can edit the existing card or create a brand new card. User is prompted with Snackbar messages for input validation
to ensure no field is set to a null or empty value. A Toast message is shown when user successfully edits an existing card
or creates a new card (i.e. saves the changes).
